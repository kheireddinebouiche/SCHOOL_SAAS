Informations sur les actions possibles
Modification

Permet de modifier les détails de l'examen (date, heure, salle, surveillant, etc.). Ce bouton est désactivé pour les examens terminés.

Impression

Permet d'imprimer les détails de l'examen ou les documents associés.

Suppression

Permet de supprimer l'examen de la planification. Ce bouton est désactivé pour les examens terminés.

Validation

Permet de valider l'examen après sa réalisation. Ce bouton est désactivé pour les examens déjà terminés.

Résultat

Permet d'accéder aux résultats de l'examen. Ce bouton est activé uniquement pour les examens terminés.
