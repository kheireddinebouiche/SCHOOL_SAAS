Guide Interactif & Aide : Configurer vos modèles de bulletins / PV

Découvrez comment exploiter toutes les fonctionnalités avancées des modèles (blocs, pondérations, moyennes, sous-notes)

1 Création de la structure globale du modèle

Un modèle de bulletin regroupe l'ensemble des règles de calcul de notes, des pondérations, et de la disposition visuelle pour une formation donnée ou pour l'ensemble de l'établissement.

  • Libellé : Donnez un nom clair (ex: "Semestre 1 - BTS").
  • Formation : Liez-le à une formation spécifique pour qu'il s'applique uniquement à ses classes, ou laissez vide pour un modèle global.
  • Modèle par défaut : Si vous l'activez, ce modèle sera utilisé automatiquement par toutes les classes qui n'ont pas de modèle spécifique.
Lancement rapide

Cliquez sur "Nouveau modèle" à droite pour démarrer !

2 Gestion des Blocs de Notes

Les blocs de notes permettent de segmenter et structurer visuellement les types d'évaluations sur le bulletin et le PV de délibération.

  • Création : Cliquez sur "Gérer les Blocs" pour définir vos blocs (ex: "Contrôle Continu", "Examen", "Rattrapage").
  • Affichage PV (Délibération) : Cochez cette case pour afficher la moyenne de ce bloc dans le tableau des délibérations.
  • Affichage Bulletin : Cochez cette case pour inclure les détails des notes de ce bloc sur le bulletin individuel de l'étudiant.
Structure Visuelle

Les blocs trient et catégorisent proprement vos formules de notes.

3 Types de Notes & Coefficients (Pondération)

Ce sont les évaluations concrètes qui reçoivent des notes (ex: CC1, CC2, Examen, Rattrapage).

  • Code unique : Un code court (ex: CC1, EXAM) requis pour les formules.
  • Note Maximale : La note plafond (ex: 20.0).
  • Coefficient (Pondération) : Le coefficient multiplicateur affecté à cette note lors des calculs de moyennes.
  • Ordre d'affichage : Un chiffre entier pour ordonner les colonnes de gauche à droite sur les grilles de saisie et de délibération.
Pondération Flexible

Chaque note a sa propre importance définie par son coefficient.

4 Notes Calculées & Dépendances

Permet d'automatiser le calcul des moyennes (ex: calculer la Moyenne de Contrôle Continu à partir de CC1 et CC2).

  • "Est calculée" : Cochez cette case pour empêcher la saisie manuelle et activer le calcul automatique.
  • Formules disponibles :
    - MOYENNE : Calcule la moyenne pondérée par les coefficients.
    - SOMME : Fait la somme brute des notes enfants.
  • Liaison des dépendances : Après avoir créé la note calculée, cliquez sur le bouton "Gérer les dépendances" (icône arborescence) et ajoutez les notes enfants qui entrent dans le calcul.
Formules Automatiques

Le système calcule instantanément les moyennes à partir des dépendances reliées !

5 Options Avancées (Sous-notes, Rattrapages, Rachats)

Le système gère des mécanismes d'évaluations complexes pour répondre à tous les règlements pédagogiques :

  • A des sous-notes (ex: S1, S2...) : Cochez cette option uniquement si une note est constituée de sous-évaluations (ex: 3 interrogations écrites sous CC1).
    Astuce cruciale : Ne cochez JAMAIS cette option pour les moyennes calculées (comme `MCC`) ou les examens simples (`EXAM`), sinon le tableau de délibération affichera une colonne inutile nommée S1.
  • Note de Rattrapage : Cochez cette case pour que cette note soit automatiquement comparée à la moyenne initiale et ne remplace celle-ci que si elle est supérieure (ou selon vos règles de délibération).
  • Rachat de crédit (Pédagogique) : Cochez cette case pour configurez une note qui permet de racheter des crédits ou compenser des modules spécifiques.
Règles Spéciales

Configurez finement vos rattrapages et sous-notes en quelques clics.

Règle de calcul de la moyenne générale :

Toutes les notes d'un étudiant sont d'abord normalisées sur 20, puis pondérées avec le coefficient saisi :

Moyenne = Σ( (Note / Max) × 20 × Coefficient ) / Σ( Coefficient )

Exemple concret :

  • Examen (Coef 1.5, Max 20) : Note obtenue 15/20 → 15 × 1.5 = 22.5
  • TP (Coef 0.5, Max 20) : Note obtenue 10/20 → 10 × 0.5 = 5.0
  • Calcul de la Moyenne : (22.5 + 5.0) / (1.5 + 0.5) = 13.75/20
