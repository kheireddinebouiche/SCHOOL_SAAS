Guide d'utilisation

Découvrez comment utiliser l'espace Inter-Etablissements

Messagerie instantanée

Sélectionnez un institut à gauche pour échanger des messages en direct. Utilisez le bouton trombone dans la zone de texte pour joindre des documents à vos messages.

Espace Global

L'Espace Global est un canal commun. Les messages et documents déposés ici sont visibles par tous les établissements de votre réseau !

Dossiers Partagés

Le panneau droit permet d'organiser vos fichiers dans une arborescence de dossiers. Cliquez sur "Nouveau dossier" pour structurer vos archives partagées.

Uploader un fichier

Dans les Dossiers Partagés, cliquez sur "Uploader un fichier" pour ajouter un document directement dans le dossier actuel, indépendemment du chat.
